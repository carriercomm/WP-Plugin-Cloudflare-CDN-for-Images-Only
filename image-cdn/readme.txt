Replaces image urls from normal upload directory with the path of cdn

== Description ==

Speed image loading by grabbing them from a cdn

It's pretty simple to run, just drop it in the plugin directory and activate that's all it takes.

== Installation ==

1. Upload files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure you have cdn-images.yourdomain.com pointed to your root directoryr
